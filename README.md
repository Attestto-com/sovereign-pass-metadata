# Sovereign Pass NFT Metadata

On-chain metadata for Sovereign Pass identity credentials.

These are **soulbound (non-transferable)** tokens issued after KYC verification.

## Token Details

| Property | Value |
|----------|-------|
| **Name** | Sovereign Pass |
| **Symbol** | SVPASS |
| **Standard** | Solana Token-2022 |
| **Extension** | NonTransferable (Soulbound) |

## Use Cases

- Identity verification
- Selective disclosure of verified attributes
- Zero-knowledge proofs (age, country, sanctions status)
- Cross-platform credential portability
